# Внешние турниры — новая колонка в рейтинге тренеров

**Дата:** 2026-04-20
**Статус:** утверждён

## Контекст

8 из 21 тренера участвуют во внешних двухмесячных турнирах (организованных другими школами/федерациями). У каждого тренера от 1 до 3 команд. Игры проходят по выходным.

Результаты внешних турниров — показатель работы тренера, который нужно отразить в рейтинге.

## Данные

### Источник

Одна новая колонка в Google Таблице `dashboard_data`:

| Колонка | Тип | Описание |
|---------|-----|----------|
| `Турниры_баллы` | число | Сумма баллов тренера за последние выходные |

### Правила подсчёта (выполняет пользователь вручную)

Формат с 2 играми за выходные: победа = 0.5, ничья = 0.5, поражение = 0.
Формат с 1 игрой за выходные: победа = 1, ничья = 0.5, поражение = 0.

Баллы всех команд тренера за выходные складываются. Пользователь вычисляет итог и вносит одно число в ячейку.

### Обновление

Данные перезаписываются каждую неделю. В ячейке всегда результат только за последние выходные (не накопительный).

Тренеры без внешних турниров — 0 или пусто.

## Рейтинг

### Нормализация

Как у остальных категорий: лучший результат среди всех тренеров = 5 баллов, остальные пропорционально.

```
score = (raw / max_raw) * 5
```

### Отображение

- Новая колонка **«Турниры»** в рейтинговой таблице
- Позиция: после «Сборные», перед «Контент»
- Сортировка по клику — как у остальных колонок

### Формула итого

```
total = camp + merch + cup + league + teams + tournaments + content - penalty
```

## Страница тренера

Новый блок с текущими баллами за турниры (сырое число из таблицы).
